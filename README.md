# Drunken Best Man EFSM

A C# console project that will model a turn-based Extended Finite State Machine game.

The project is currently in initial setup. The architecture will be layered, and narrative texts will later be stored in XML. Game logic is not implemented yet.

The initial EFSM model is documented in [docs/efsm-model.md](docs/efsm-model.md).
The intended layered architecture is documented in [docs/architecture.md](docs/architecture.md).
